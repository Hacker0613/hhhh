Введение

Прежде чем начать!

Добро пожаловать в основы Android с Compose! Этот курс предназначен для тех, кто хочет научиться создавать приложения для Android на языке программирования Kotlin, используя последние рекомендации Google. Мы будем использовать Jetpack Compose - современный способ разработки красивых пользовательских интерфейсов (UI) для ваших Android-приложений.

В ходе обучения вы будете работать с пошаговыми лабораторными работами, предоставляющими инструкции по программированию приложения или его частей. Если вы предпочитаете визуальное обучение, некоторые из этих лабораторных работ снабжены видеороликами, демонстрирующими код. Также вас ждут практические задания для закрепления материала и проекты для самостоятельного экспериментирования и углубленного изучения.

Обучение - это индивидуальный опыт, и у всех он разный. В зависимости от вашего уровня знаний и опыта предлагаются различные подходы к освоению курса.

Новички в программировании:

Начните с начала курса и последовательно пройдите все этапы обучения. Концепции программирования и Android взаимосвязаны, поэтому важно осваивать материал последовательно. Рекомендуется также выполнять практические задачи и проекты для укрепления навыков начинающего разработчика.

Опытные разработчики:

Если у вас уже есть опыт программирования, вы можете пропустить часть материала, чтобы более быстро пройти курс. В начале каждого блока представлены кодовые таблицы для необходимых концепций Kotlin, необходимых для создания приложений. Если вы уверены в этих разделах, пропустите их и переходите к материалу по созданию приложений.

Разработчикам Android может быть интересен раздел по Compose, представленный в этом курсе. Compose - современный инструментарий Android для создания пользовательских интерфейсов, упрощающий и ускоряющий процесс разработки. Первая половина курса наиболее актуальна для изучения пользовательских интерфейсов, дизайна материалов и навигации. Альтернативно, вы можете изучить курс Jetpack Compose для разработчиков Android, который предоставляет статьи, видео и кодовые таблицы по ключевым темам для разработчиков, переходящих на Compose.
